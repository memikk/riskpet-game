# RiskPet

A Tamagotchi-inspired risk-to-earn game prototype built with React, Three.js, and Solidity for the Base ecosystem.

## Overview

RiskPet explores the idea of combining digital pet care with competitive risk mechanics. Players manage a virtual creature, improve its state, and connect gameplay outcomes to blockchain-based systems and smart contract logic.

## Highlights

- 3D interactive pet interface
- rarity-based pet system
- booster mechanics and progression design
- wallet connection flow
- smart contract integration with Solidity and Hardhat
- Base-oriented Web3 deployment structure

## Tech Stack

- React
- TypeScript
- Vite
- Three.js
- React Three Fiber
- Wagmi
- RainbowKit

## Running Locally

```bash
npm install
npm run dev
```

For contract development:

```bash
cd hardhat
npm install
npx hardhat compile
```

## Environment Setup

Use `.env.example` as the starting point for local configuration. Do not commit real private keys or production secrets.
